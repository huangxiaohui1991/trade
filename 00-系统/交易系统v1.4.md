# A股交易系统 v1.4

> 核心框架：行业逻辑 + 基本面 + 技术面 + 主力资金，四维评分选股

---

## 📊 选股系统

### 四维评分模型（满分10分，≥7分通过）

| 维度 | 分值 | 评分标准 |
|------|------|---------|
| **行业逻辑** | 0-3分 | 政策/主题/周期/格局/护城河 |
| **基本面** | 0-3分 | ROE/毛利率/营收增速/现金流 |
| **技术面** | 0-2分 | 趋势/均线/量价结构 |
| **主力资金** | 0-2分 | 主力净流入/超大单/大单 |

**通过标准**：总分 ≥ 7分，单项不得为0

### 板块选择原则
- 政策扶持（10分）+ 周期拐点（9分）+ 格局清晰（8分）优先
- 规避：产能过剩/强周期/高污染/明显天花板

## 💰 仓位管理

| 参数 | 数值 |
|------|------|
| 单票上限 | 20%（总账户）|
| 单账户上限 | 60%（总账户）|
| 单笔金额 | 2-3万 |
| 持仓数量 | 3-5只 |
| 最小建仓 | 1/3仓 |

**公式**：建仓股数 = MIN(总资金×20%×0.33, 总资金×10%, 30000÷股价)

## ✅ 买入规则

### 信号确认（缺一不可）
1. **板块效应**：板块内有明显联动效应（≥3只）
2. **2日确认**：首仓需等回调后第2日确认信号
3. **量价配合**：涨幅 > 5% + 放量（量能 > 前高量 1.5倍）
4. **不在高位**：非连续拉升后第3板

### 买入限制（任一触发暂停买入）
- 连续止损2笔 → 停止3天
- 单日回撤 > 3% → 停止3天
- 大盘环境差（三大指数全绿）→ 停止买入

### 建仓流程
```
观察池筛选 → 追涨信号确认 → 试仓1/3 → 
次日加仓1/3（站稳确认）→ 再次日1/3（全仓）
```

## 🚨 卖出规则

### 三重止损
| 层级 | 条件 | 动作 |
|------|------|------|
| 第一层 | 浮亏 > 4% | 减仓1/3 |
| 第二层 | 跌破3日均线 | 清仓 |
| 第三层 | 浮亏 > 7% | 绝对止损（无条件清仓）|

### 时间止损
- 建仓后 **15个交易日** 内未盈利（振幅<4%）→ 主动减仓
- 理由：市场不认可=逻辑错误，给予足够时间但不死扛

### 分批止盈（回撤追踪）
| 条件 | 动作 |
|------|------|
| 涨幅 > +15% | 减仓1/3（移动止损起点确立）|
| 从高点回撤 > 5% 或 累计涨幅 > +30% | 再减1/3（累计卖2/3）|
| 跌破20日线 或 从高点回撤 > 8% | 清仓（剩余1/3）|

> 核心逻辑：让利润奔跑，用回撤追踪代替静态价格目标

## 🏊 泳池管理

### 核心池
- 入池条件：评分 ≥ 7分 + 主力首次表态
- 持仓上限：总仓位 60%
- 持仓数量：3只
- 淘汰条件：连续2笔止损 OR 逻辑破坏

### 观察池
- 入池条件：板块内有联动，关注但未建仓
- 持仓上限：总仓位 20%（可选）
- 持仓数量：3只
- 升级条件：核心池标的出现买入信号

## 📋 每日检核

```
1. 持仓检查（开盘前30分钟）
   - 是否有止损信号？
   - 是否有止盈机会？
   - 是否触发时间止损？

2. 观察池扫描
   - 有无新信号出现？
   - 板块联动是否强化？

3. 大盘环境判断
   - 三大指数状态（涨跌家数）
   - 情绪周期定位

4. 记录日志
   - 持仓变动及理由
   - 观察池变化
   - 当日心得
```

## 🏊 遗留仓位管理

**定义**：新系统启动前已持有的仓位，独立于新系统规则。

| 规则 | 说明 |
|------|------|
| **不补仓** | 遗留仓位绝不加仓，摊薄成本 |
| **不合并** | 不与新系统混合计算仓位 |
| **分批减仓** | 优先反弹卖出一部分，降低风险敞口 |
| **绝对止损** | 港股遗留仓位：-15%无条件清仓（A股规则7%过严不适用于港股高波幅）|
| **移动止损** | 每反弹+5%上调止损价至成本价 |

## ⚠️ 风控红线

1. **不逆势**：大盘全绿时不建仓
2. **不重仓**：单票 ≤ 20%
3. **不侥幸**：浮亏 > 7% 必须清仓
4. **不频繁**：连续2笔止损停止3天
5. **不接刀**：连续下跌后不抄底
6. **不追高**：连续拉升第3板不追
