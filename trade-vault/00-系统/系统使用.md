# 系统使用

所有命令通过 `bin/trade` 调用。

## CLI 命令

```bash
# 数据库
bin/trade db init          # 初始化
bin/trade db migrate       # 运行 migration
bin/trade db status        # 查看状态

# 配置
bin/trade config freeze    # 冻结当前配置
bin/trade config history   # 版本历史

# 评分
bin/trade score 002138     # 单股四维评分

# 持仓
bin/trade status           # 持仓概览

# Pipeline（带幂等检查，同一天不会重复执行）
bin/trade run-pipeline morning    # 盘前摘要
bin/trade run-pipeline evening    # 收盘报告
bin/trade run-pipeline scoring    # 核心池评分
bin/trade run-pipeline weekly     # 周报

# 事件查询
bin/trade events query --type score.calculated
bin/trade events count --type order.filled

# 运行记录
bin/trade runs list
bin/trade runs failed

# MCP Server
bin/trade mcp              # 启动 MCP Server (stdio)
```

## Cron 调度

```
25 8  * * 1-5  bin/trade run-pipeline morning
35 15 * * 1-5  bin/trade run-pipeline evening
40 15 * * 1-5  bin/trade run-pipeline scoring
0  20 * * 0    bin/trade run-pipeline weekly
```

## Pipeline 流程

| pipeline | 做什么 | 输出 |
|---|---|---|
| morning | 大盘信号 → 持仓风控 → 核心池 → 今日决策 | Obsidian + Discord |
| scoring | 核心池抓数据 → 评分 → 更新池子 | Obsidian + Discord |
| evening | 大盘信号 → 持仓风控 → 收盘报告 | Obsidian + Discord |
| weekly | 统计交易 → 胜率/盈亏比 → 周报 | Obsidian + Discord |
