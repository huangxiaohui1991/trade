# 每日流程

> 版本：v1.0
> 更新日期：2026-04-13

---

## 一、时间轴总览

| 时间 | 任务 | 命令 |
|------|------|------|
| 08:25 | 盘前准备 | `bin/trade run morning` |
| 11:55 | 午休检查 | `bin/trade run noon` |
| 15:00 | 收盘快照 | `bin/trade run evening` |
| 15:40 | 核心池评分 + 分池 | `bin/trade run scoring` |
| 任意时间 | 选股流水线 | `bin/trade run screener --pool all --universe tracked` |
| 任意时间 | 舆情监控 | `bin/trade run sentiment --dry-run` |
| 周五 15:30 | 周复盘 | `bin/trade run weekly` |

---

## 二、各时段详解

### 08:25 盘前（morning）

- 获取大盘指数状态（沪深300、上证、创业板）
- 读取持仓股最新行情
- 检查持仓股是否触发止损/止盈条件
- 生成盘前简报，写入 `02-运行/信号快照/今日信号.md`

```bash
bin/trade run morning
```

### 11:55 午休（noon）

- 检查上午交易时段是否有新信号触发
- 确认上午条件单执行情况
- 更新午盘状态

```bash
bin/trade run noon
```

### 15:00 收盘（evening）

- 记录收盘价，更新持仓盈亏
- 评估是否有止损/止盈信号收盘触发
- 生成收盘快照，写入 `02-运行/信号快照/收盘快照.md`

```bash
bin/trade run evening
```

### 15:40 核心池评分（scoring）

- 对核心池 + 观察池所有持仓股票重新评分
- 执行自动分池规则（晋级/降级/移出）
- 写入池子快照 `01-状态/核心池.md`、`01-状态/观察池.md`
- 生成池子调整建议 `04-决策/筛选结果/池子调整建议_*.md`

```bash
bin/trade run scoring
```

> 建议使用 Cron 自动化定时触发：
> `57 15 * * 1-5 cd /path/to/trade && bin/trade run scoring`

### 每日选股（screener）

- 按需运行，不强制每日执行
- 推荐工作日收盘后执行

```bash
# 已跟踪池（稳定，推荐）
bin/trade run screener --pool all --universe tracked

# 全市场（依赖妙想 API，建议配合超时和重试）
bin/trade orchestrate market_scan
```

### 舆情监控（sentiment）

- 扫描持仓股和候选股最新新闻/公告
- 检测业绩暴雷、重大事项等风险信号
- 默认 dry-run，不推送 Discord

```bash
bin/trade run sentiment --dry-run   # 只扫描不上报
bin/trade run sentiment             # 扫描并推送 Discord
```

### 周五收盘 周复盘（weekly）

- 统计本周交易数据（盈利/亏损、胜率）
- 汇总持仓股本周走势
- 生成周报，写入 `03-分析/周复盘/YYYY-QN.md`

```bash
bin/trade run weekly
```

## 三、Workflow 编排（推荐）

对于自动化 Agent（Hermes/OpenClaw），使用 `orchestrate` 命令：

```bash
# 盘前简报
bin/trade orchestrate morning_brief

# 午盘检查
bin/trade orchestrate noon_check

# 收盘复盘（收盘 + 评分）
bin/trade orchestrate close_review

# 周复盘
bin/trade orchestrate weekly_review

# 已跟踪池扫描（稳定）
bin/trade orchestrate tracked_scan

# 全市场扫描（高风险，依赖外部 API）
bin/trade orchestrate market_scan
```

查看所有可用 workflow：

```bash
bin/trade workflows
```

## 四、状态查看

```bash
# 查看今日状态（持仓、信号、决策）
bin/trade status today

# 健康检查（数据源、API、目录权限）
bin/trade doctor

# 查看当前池子快照
cat trade-vault/01-状态/核心池.md
cat trade-vault/01-状态/观察池.md

# 查看今日决策
cat trade-vault/04-决策/今日决策.md
```

## 五、Cron 自动化建议

```bash
# 08:25 盘前
25 8 * * 1-5 cd /path/to/trade && bin/trade orchestrate morning_brief >> logs/morning.log 2>&1

# 11:55 午休
55 11 * * 1-5 cd /path/to/trade && bin/trade orchestrate noon_check >> logs/noon.log 2>&1

# 15:00 收盘
5 15 * * 1-5 cd /path/to/trade && bin/trade orchestrate close_review >> logs/evening.log 2>&1

# 每周五收盘复盘
30 15 * * 5 cd /path/to/trade && bin/trade orchestrate weekly_review >> logs/weekly.log 2>&1
```
