## 一、入场信号（买入触发）

### 基础触发条件（三条件同时满足）

| 条件 | 参数 | 说明 |
|------|------|------|
| MA 金叉 | MA10 上穿 MA20 | 趋势由弱转强 |
| 量比 | > 1.5 | 放量确认 |
| RSI | < 70 | 防追高接盘 |

### C 版新增过滤

```yaml
entry_signal:
  deviation_max: 10.0          # 乖离率 < 10%
  require_bullish_candle: true  # 金叉当日必须收阳线
```

### 慢牛补充信号（回踩买入）

```yaml
pullback:
  enabled: true
  ma_support: 20              # 回踩 MA20 不破
  volume_confirm: true        # 次日放量确认
  require_bullish_candle: true # 次日阳线
```

## 二、仓位管理

### 全局仓位限制

```yaml
risk:
  portfolio:
    daily_loss_limit_pct: 0.03        # 单日亏损上限 3%
    consecutive_loss_days_limit: 2    # 连续亏损 2 天后冷却
    cooldown_days: 2                  # 冷却期 2 天
    max_single_position_warn_pct: 0.25 # 单票仓位警告线 25%
    max_sector_exposure_warn_pct: 0.40 # 行业仓位警告线 40%
    max_correlation_group_exposure_warn_pct: 0.50 # 相关股组仓位 50%
  position:
    total_max: 0.60   # 总仓位上限 60%
    single_max: 0.20   # 单票仓位上限 20%
    weekly_max: 2      # 每周最多买 2 只
    holding_max: 4     # 最多同时持有 4 只
```

## 三、风控参数（双轨模式）

系统根据股票波动特征自动判定为慢牛股或题材趋势股，执行不同风控规则。

### 慢牛成长股

特征：日波动 < 2%，RSI 常年 50-65，MA20 斜率 > 0.5%/天

```yaml
risk:
  slow_bull:
    stop_loss: 0.08           # -8% 止损
    absolute_stop_ma: 60      # 或跌破 MA60
    take_profit: none         # 不主动止盈（让利润奔跑）
    exit_ma: 20               # 收盘跌破 MA20 离场
    time_stop_days: 30        # 30 个交易日不涨才审视
```

### 题材趋势股

特征：日波动 > 3%，RSI 易冲高至 75+，MA20 斜率 > 2%/天

```yaml
risk:
  momentum:
    stop_loss: 0.08                  # -8% 止损
    stop_loss_anchor: entry_day_low  # 或跌破买入日最低价
    trailing_stop: 0.10              # 移动止盈：最高点回撤 -10%
    trailing_ma_cross:               # 或 MA5 死叉 MA10
      fast: 5
      slow: 10
    exit_ma: 20                      # 收盘跌破 MA20 离场
    time_stop_days: 15               # 15 个交易日不创新高即警惕
```

### 风格自动切换（保险丝）

当慢牛股出现以下情况，自动切换为题材趋势股风控：

```yaml
style_switch:
  triggers:
    single_day_surge_pct: 0.07      # 单日涨幅 > 7%
    rsi_overheat_threshold: 75       # RSI > 75
    rsi_overheat_consecutive_days: 3  # 连续 3 日
  action: switch_to_momentum
```

## 四、卖出规则

| 触发条件 | 适用类型 | 操作 |
|----------|----------|------|
| -8% 止损 | 全部 | 无条件卖出 |
| 跌破 MA60 | 慢牛股 | 卖出 |
| 移动止盈回撤 -10% | 题材股 | 卖出 |
| MA5 死叉 MA10 | 题材股 | 卖出 |
| 收盘跌破 MA20 | 全部 | 卖出 |
| 30天/15天不涨 | 慢牛/题材 | 审视是否离场 |

## 五、实盘操作

实盘操作通过条件单指令执行，不在 CLI 中直接下单。

```bash
# 查看当前组合状态
bin/trade state orders --scope paper_mx

# 查看持仓
bin/trade status today

# 模拟盘买入选股
bin/trade mx run mx.moni.buy --stock-code 300938 --price 19.50 --shares 100

# 模拟盘卖出
bin/trade mx run mx.moni.sell --stock-code 300938 --price 21.00 --shares 100

# 取消订单
bin/trade order cancel --code 300938 --scope paper_mx

# 查看风控状态
bin/trade backtest run --start 2026-01-01 --end 2026-04-13 --scope cn_a_system
```
