## 二、系统架构

```
bin/trade
├── run/              # 每日任务
│   ├── morning       # 盘前
│   ├── noon         # 午休
│   ├── evening       # 收盘
│   ├── scoring       # 核心池评分
│   ├── screener      # 选股流水线
│   └── weekly        # 周复盘
├── backtest/         # 回测引擎
├── workflows/        # 编排任务
└── ...

数据源优先级：妙想 API → AKShare → 新浪/东财
唯一事实来源：Obsidian Vault (trade-vault/)
```

## 四、快速开始

### 日常选股

```bash
# 核心池评分（收盘15:40自动）
bin/trade run scoring

# 选股流水线（全市场）
bin/trade run screener --pool all --universe tracked

# 查看今日决策
cat trade-vault/04-决策/今日决策.md
```

### 配置修改

```bash
# 策略参数
vim config/strategy.yaml

# 股票池/黑名单
vim config/stocks.yaml

# 妙想选股条件
vim config/strategy.yaml  # screening.mx_query
```
