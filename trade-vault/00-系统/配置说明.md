## 一、配置文件一览

| 文件 | 用途 |
|------|------|
| `config/strategy.yaml` | 策略参数（选股条件、入场信号、风控、评分权重） |

## 二、strategy.yaml 详解

### 2.1 粗筛条件（screening）

```yaml
screening:
  mx_query: "站上20日均线，ROE大于8%，成交额超5000万，股价5-200元，市盈率TTM低于100，剔除ST"
  mx_select_type: "A股"
  market_scan_limit: 30        # 全市场最多送评 30 只
  candidate_cache_ttl_hours: 24 # 候选缓存有效期（小时）
  fallback_filters:
    min_price: 5               # 最低价
    max_price: 200             # 最高价
    min_amount: 50_000_000      # 最低成交额（5000万）
    max_pe_ttm: 100            # PE(TTM) 上限
    exclude_st: true            # 剔除 ST
    ma20_trend_up: true         # MA20 趋势向上（新增）
```

### 2.2 入场信号（entry_signal）

> **统一配置原则**：entry_signal 是回测引擎和实盘评分的共同配置源。
> backtest_presets 中的字段会覆盖此处值，形成 override 差量。
> 回测引擎无硬编码默认值，所有回退值均来自 strategy.yaml。

```yaml
entry_signal:
  entry_mode: hybrid  # hybrid（默认）/ strict / trend_follow / score_only
  ma_cross:
    fast: 10       # MA10
    slow: 20       # MA20
    type: golden_cross  # 金叉
  volume_ratio_min: 1.5    # 量比最小 1.5
  rsi_max: 70             # RSI 上限（防追高）
  rsi_period: 14
  deviation_max: 10.0     # 乖离率上限 10%（C版）
  require_bullish_candle: true  # 金叉当日必须收阳（C版）

  pullback:              # 回踩买入（可选）
    enabled: true
    ma_support: 20       # 回踩 MA20
    volume_confirm: true
    require_bullish_candle: true
```

### 2.3 风格判定（style_classifier）

```yaml
style_classifier:
  slow_bull:            # 慢牛成长股
    daily_volatility_max: 0.02   # 日波动上限 2%
    rsi_range: [50, 65]          # RSI 区间
    ma20_slope_min: 0.005        # MA20 日斜率下限 0.5%
    ma20_slope_lookback_days: 10
  momentum:             # 题材趋势股
    daily_volatility_min: 0.03   # 日波动下限 3%
    rsi_high_threshold: 75        # RSI 可冲高至 75+
    ma20_slope_min: 0.02         # MA20 日斜率下限 2%
```

### 2.5 评分权重（scoring）

```yaml
scoring:
  weights:
    technical: 3      # 技术面权重
    fundamental: 2    # 基本面权重
    flow: 2           # 资金流权重
    sentiment: 3      # 舆情权重
  thresholds:
    buy: 6.5    # 买入阈值
    watch: 5    # 观察阈值
    reject: 4   # 拒绝阈值
  veto:
    - below_ma20            # 股价 < MA20
    - limit_up_today        # 今日涨停
    - consecutive_outflow   # 连续资金流出
    - red_market            # 大盘下跌
    - earnings_bomb         # 业绩暴雷
    - ma20_trend_down       # MA20 趋势向下
```

### 2.6 池子管理（pool_management）

```yaml
pool_management:
  watch_min_score: 5           # 观察池最低分
  promote_min_score: 6.5       # 晋级核心池分数
  promote_streak_days: 2       # 晋级需连续天数
  demote_max_score: 5          # 降级分数线
  demote_streak_days: 2        # 降级需连续天数
  remove_max_score: 4          # 移出分数线
  remove_streak_days: 2        # 移出需连续天数
  veto_immediate_demote: true  # 一票否决立即降级
  add_to_watch_streak_days: 1  # 入观察池需连续天数
```

### 2.7 大盘择时（market_timer）

```yaml
market_timer:
  green_days: 3         # 连续上涨 3 天提示谨慎
  red_days: 5            # 连续下跌 5 天可能见底
  clear_days_ma60: 15    # 站上 MA60 超过 15 天信号强
```

### 2.8 回测 Preset

> **配置模型**：backtest_presets 为差量 override，不填的字段自动继承 strategy.yaml 基础配置。
> 因此每个 preset 只写"与基础值不同"的参数，无需重复已对齐的值。

```yaml
backtest_presets:
  aggressive_high_return:
    description: "高收益优先，接受更高交易频次"
    buy_threshold: 6.0            # 基础 6.5 → 放宽到 6.0
    momentum_trailing_stop: 0.10   # 基础 0.10（显式，无变化）
    momentum_time_stop_days: 10    # 基础 15 → 缩短到 10

  保守验证C:
    description: "乖离率<10% + 收阳线 + trailing10% + time_stop15天，稳健首选"
    buy_threshold: 6.5            # 同基础值，显式强调
    momentum_trailing_stop: 0.10  # 同基础值
    momentum_time_stop_days: 15   # 基础 15（已在 strategy.yaml，显式对齐）
    entry_trend_rsi_max: 70      # 基础 70
    entry_pullback_rsi_max: 70    # 基础 70
    entry_pullback_volume_ratio_min: 1.5  # 同基础值
    entry_deviation_max: 10.0     # 基础 10.0
    entry_require_bullish_candle: true    # 同基础值

  大盘择时增强版:
    description: "金叉收阳 + 乖离率<12% + 两市成交>6000亿，高胜率但交易频次低"
    buy_threshold: 6.0            # 放宽到 6.0
    momentum_trailing_stop: 0.18   # 基础 0.10 → 放宽止盈
    momentum_time_stop_days: 40   # 基础 15 → 延长到 40
    entry_trend_rsi_max: 85       # 基础 70 → 放宽
    entry_pullback_rsi_max: 85    # 基础 70 → 放宽
    entry_pullback_volume_ratio_min: 1.0  # 基础 1.5 → 放宽
    entry_deviation_max: 12.0     # 基础 10.0 → 放宽
    entry_require_bullish_candle: true   # 同基础值
```

## 四、环境变量（.env）

```bash
# 妙想 API（选股、舆情、自选股同步）
MX_API_KEY=your_api_key_here
MX_BASE_URL=https://api.example.com

# Discord Webhook（推送信号通知）
DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/...

# 数据源（通常无需修改）
AKSHARE_FALLBACK=true
```

## 五、修改配置的推荐流程

1. 先用回测验证参数效果：
   ```bash
   bin/trade backtest validate-single --code 300938 --start 2026-01-01 --end 2026-04-13
   ```

2. 参数 sweep 找到最优区间：
   ```bash
   bin/trade backtest sweep --start 2026-01-01 --end 2026-04-13 --buy-thresholds 6.0,6.5,7.0
   ```

3. 确认无误后再修改 `strategy.yaml`
