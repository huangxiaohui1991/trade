# trade-vault

这是交易系统的 Obsidian 内容区，和代码仓放在同一个 git 仓库里，但目录分开。

目录约定：

- `00-系统`：仪表盘、说明、模板
- `01-状态`：持仓、账户、池子
- `02-运行`：每日日志、模拟盘、信号快照、当日输出
- `03-分析`：周复盘、月复盘、专题分析、策略体检
- `04-决策`：今日决策、候选池、筛选结果、个股解释

使用约定：

- Python 代码默认从仓库根的 `trade-vault/` 读取内容
- 如需改路径，可设置 `AStockVault` 或修改 `config/paths.yaml`
- Obsidian 建议直接打开本目录作为 vault 根
