# 选股分析

用户请求选股时执行，或作为定期补充核心池的流程。

## 流程

1. 调用 `trade_screener` 执行选股筛选
   - 默认条件：站上20日均线，ROE>8%，成交额>5000万，股价5-200元，PE<100，剔除ST
   - 用户可自定义 query
2. 对筛选结果调用 `trade_score_batch` 批量评分
3. 按总分降序排列，标记买入/观察/规避
4. 对评分 ≥6.5 的股票建议加入观察池

## 输出格式

```
🔍 选股结果 — {日期}

筛选条件：{query}
命中 {N} 只，评分后：

| # | 股票 | 代码 | 总分 | 风格 | 建议 |
|---|------|------|------|------|------|
| 1 | {名称} | {代码} | {分数} | {慢牛/题材} | {建议} |

## 推荐关注（评分≥6.5）
- {股票名}（{代码}）评分 {分数}，建议加入观察池
```

## 规则

- 一票否决的直接标记 ❌，不推荐
- 已在核心池/观察池的标记"已在池中"
- 最多展示前 20 只
