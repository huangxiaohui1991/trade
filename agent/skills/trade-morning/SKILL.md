# 盘前摘要

每个交易日 08:25 执行。

## 流程

1. 调用 `trade_market_signal` 获取大盘择时信号
2. 调用 `trade_portfolio` 获取当前持仓
3. 调用 `trade_pool_status` 获取核心池状态
4. 对核心池中评分过期（>1天）的股票调用 `trade_score_stock` 刷新评分
5. 如果有持仓，对每只持仓调用 `trade_check_risk` 检查风控
6. 汇总以上信息，生成盘前摘要

## 输出格式

```
📊 盘前摘要 — {日期}

## 大盘信号
{信号}: {上证/深证/创业板 涨跌幅}

## 持仓（{N}只）
- {股票名} {股数}股 成本¥{成本} | 风控: {正常/预警}

## 核心池（{N}只）
- {股票名} 评分{分数} {✅/🟡/❌}

## 交易计划
- 本周已买 {N}/{上限} 次
- {买入建议 / 观望 / 禁止新开仓}
```

## 规则

- 大盘 RED/CLEAR 时，明确提示"禁止新开仓"
- 持仓有止损预警时，突出显示并建议操作
- 核心池评分 ≥7 且大盘 GREEN 时，标记为"可买入"
