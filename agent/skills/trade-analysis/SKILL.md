# 个股深度分析

用户问"帮我看看 XXX 能不能买"时执行。

## 流程

1. 调用 `trade_score_stock` 获取四维评分
2. 调用 `trade_market_signal` 获取大盘信号
3. 调用 `trade_score_history` 查看历史评分趋势
4. 调用 `trade_calc_position` 计算建议仓位
5. 调用 `trade_check_portfolio_risk` 检查组合风控是否允许买入
6. 综合分析，给出明确建议

## 输出格式

```
📋 {股票名}（{代码}）分析

## 四维评分：{总分}/10
- 技术面 {分数}/3：{详情}
- 基本面 {分数}/3：{详情}
- 资金流 {分数}/2：{详情}
- 舆情面 {分数}/3：{详情}

## 风格判定：{慢牛/题材}
{判定依据}

## 大盘环境：{GREEN/YELLOW/RED}
{对买入的影响}

## 历史评分趋势
{最近N天评分变化}

## 建议
{明确的买入/观察/规避建议}
{如果建议买入：仓位、止损价、止盈目标}
{如果建议观察：关注什么条件再入场}
{如果建议规避：原因}
```

## 规则

- 一票否决时直接给出"规避"建议，说明否决原因
- 大盘 RED/CLEAR 时，即使评分高也建议"等待大盘转好"
- 组合风控不允许时，说明原因（仓位已满/本周额度用完等）
- 给出具体的止损价位（基于风格：慢牛-8%/题材-8%+移动止盈10%）
