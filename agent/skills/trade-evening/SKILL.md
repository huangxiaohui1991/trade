# 收盘报告

每个交易日 15:35 执行。

## 流程

1. 调用 `trade_market_signal` 获取收盘大盘信号
2. 调用 `trade_portfolio` 获取持仓及当日盈亏
3. 对每只持仓调用 `trade_check_risk` 检查风控（止损/止盈/时间止损）
4. 调用 `trade_pool_status` 获取核心池变化
5. 调用 `trade_trade_events` 获取今日成交记录
6. 汇总生成收盘报告

## 输出格式

```
📈 收盘报告 — {日期}

## 大盘
{上证} {涨跌幅} | {深证} {涨跌幅} | {创业板} {涨跌幅}

## 持仓盈亏
- {股票名} {股数}股 成本¥{成本} 现价¥{现价} 盈亏{+/-百分比}

## 风控事件
- {止损/止盈触发详情，无则"无触发"}

## 今日成交
- {买入/卖出记录，无则"无成交"}

## 明日计划
- {基于风控和评分的操作建议}
```

## 规则

- 风控触发时用 ⚠️ 标记，止损用 🔴，止盈用 🟢
- 持仓亏损超过 5% 的突出显示
- 明日计划要具体：挂什么价位的条件单
