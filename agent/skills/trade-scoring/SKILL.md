# 核心池评分

每个交易日 15:40 执行（收盘报告之后）。

## 流程

1. 调用 `trade_score_batch` 对核心池全部股票批量评分
2. 调用 `trade_market_signal` 获取大盘信号
3. 根据评分结果判断池子变动：
   - 评分 ≥6.5 连续 2 天 → 晋升核心池
   - 评分 <5 连续 2 天 → 降级到观察池
   - 评分 <4 连续 2 天 → 移出池子
   - 一票否决 → 立即降级
4. 对评分 ≥7 且大盘 GREEN 的股票，调用 `trade_calc_position` 计算建议仓位
5. 汇总生成评分报告

## 输出格式

```
🎯 核心池评分 — {日期}

| 股票 | 总分 | 技术 | 基本面 | 资金 | 舆情 | 风格 | 建议 |
|------|------|------|--------|------|------|------|------|
| {名称} | {分数} | ... | ... | ... | ... | {慢牛/题材} | {✅可买/🟡观察/❌规避} |

## 池子变动
- ⬆️ {股票} 晋升核心池（连续{N}天≥6.5）
- ⬇️ {股票} 降级观察池（连续{N}天<5）

## 买入建议
- {股票} 评分{分数} 建议仓位{百分比}（{股数}股 ≈ ¥{金额}）
```

## 规则

- 数据质量 degraded 的股票标记 ⚠️，不作为自动买入依据
- 一票否决的股票评分显示为 0，标记否决原因
- 舆情搜索只对三维排名前 8 名执行，其余默认 1.5 分
