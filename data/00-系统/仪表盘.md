## 本周交易统计

```dataview
TABLE date as "日期", daily_pnl as "盈亏", trades as "交易次数", mood as "情绪"
FROM #交易日志
WHERE date >= date(today) - dur(7 days)
SORT date ASC
```

## 本周买入次数

```dataview
TABLE sum(rows.trades) as "总交易次数"
FROM #交易日志
WHERE date >= date(today) - dur(7 days)
```

## 本月收益趋势

```dataview
TABLE date as "日期", total_asset as "总资产", daily_pnl as "当日盈亏"
FROM #交易日志
WHERE date >= date(today) - dur(30 days)
SORT date ASC
```

## 最近周复盘

```dataview
TABLE week as "周次", weekly_pnl as "周盈亏", buy_count as "买入次数", win_count as "盈利", loss_count as "亏损"
FROM #周复盘
SORT date DESC
LIMIT 4
```
